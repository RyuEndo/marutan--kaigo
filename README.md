# Line Bot for IRMAN

```
１．友達登録時
・あいさつ文
・クイックリプライで整形の経験はあるかを聞く

２．部位の選択
「ありがとうございます。」
・クリックリプライで部位を選んでもらう
(まぶた、鼻、口、など)

３．病院を見る基準
・反応する文章
「〇〇に悩みを持ってるのですね」
・クイックリプライで病院を選ぶにあたって特に重視する項目を選んでもらう
(値段、症例、評判、カウンセリングの質)

4.基準を選択後
・反応する文章
「〇〇を重視するのですね」
・住んでいる地域を選択してもらう(地域)
・住んでいる地域を選択してもらう(都道府県)

５．病院の紹介と絞り込み
・反応する文章
「ありがとうございます。」
・ユーザーの反応に適した病院を3つくらいバブルメッセージで紹介
・良さそうな病院を選んでもらう

６．病院の絞り込み
・反応する文章
「〇〇ですね。病院の詳細はこちらです」
・病院の詳細を送る
・クイックリプライで次の行動を選択してもらう(この病院に話を聞きに行く、他の病院を見る)

７．予約
・反応する文章(他の病院を見る、の場合は５に戻る)
・カウンセリングの予約に進む~~~~
```
